## Install

To run this repository please install [`node`](https://nodejs.org) 6 or higher.

Once cloned/pulled/downloaded install dependencies:

```shell
npm install
```

Then run the build:

```shell
npm run build
```

To run the watch while developing and spin up a local server run:

```shell
npm run watch
```
